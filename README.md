# REGISTRVM

REGISTRVM is an open-source Renaissance-inspired display typeface developed as a university project for the Type Design course at RUFA - Rome University of Fine Arts by Mattia Capomagi and Leonardo Borri.

The typeface is not a design drawn entirely from scratch. It is a typographic revival based on a historical specimen found in the Vatican Archives, with additional influence from Garamond-style old-style roman forms.

The project is still in an early stage. The current font is intended for titles, inscriptions, short display text, and historical or editorial layouts where a classical roman atmosphere is appropriate.

## Status

Current version: `1.0`

Authors: Mattia Capomagi and Leonardo Borri

Included in this repository:

- `REGISTRVM.otf`: installable OpenType font
- `REGISTRVM.glyphs`: original Glyphs source file
- `LICENSE`: project license

Current character coverage is limited. The font includes uppercase Latin letters, selected accented capitals, punctuation, symbols, and a few stylistic alternates. Lowercase Unicode characters are currently mapped to the same uppercase forms, so the typeface does not yet include a dedicated lowercase design.

## Character Set

The current source includes:

- Basic Latin capitals `A-Z`
- Selected accented letters: acute and grave variants for `A`, `E`, `I`, `O`, and `U`
- Basic punctuation and symbols, including period, comma, colon, semicolon, exclamation mark, question mark, quotes, slash, backslash, hyphen, underscore, asterisk, and ampersand
- Stylistic alternates for selected letters

This is not yet a complete text family. Contributions that expand language support, spacing, kerning, lowercase letters, numerals, additional punctuation, weights, or styles are welcome.

## Source

The editable source is `REGISTRVM.glyphs`. If you want to modify or extend the typeface, start from this file rather than editing the compiled `.otf` directly.

Suggested contribution areas:

- Draw a true lowercase set
- Add numerals
- Expand accented Latin coverage
- Improve spacing and kerning
- Add additional stylistic alternates
- Create new weights or styles
- Test the font in real layouts and report issues

## Installation

Download `REGISTRVM.otf` and install it through your operating system's font manager.

On macOS, double-click the file and choose **Install** in Font Book. On Windows, right-click the file and choose **Install**.

## Contributing

REGISTRVM is open source so that other designers, typographers, and developers can study it, improve it, and build on it.

If you contribute changes, please keep the historical character of the design in mind. The goal is not to turn REGISTRVM into a generic Garamond revival, but to preserve the personality of the Vatican specimen while making the font more usable.

Recommended workflow:

1. Fork the repository.
2. Edit `REGISTRVM.glyphs`.
3. Export a new `REGISTRVM.otf`.
4. Open a pull request describing what changed.

Please include screenshots or specimen images when proposing visual changes.
